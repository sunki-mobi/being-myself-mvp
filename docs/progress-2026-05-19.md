## 3. 코칭 prompt — Q1/Q2 역할 분리의 진화 (5/14 → 5/18)

박람회 직후 LLM이 두 질문을 거의 같은 톤으로 생성하던 문제. 5단계로 정밀화:

1. **Phase A (5/15)** — Q1은 베이스라인/이전 답변/일기 환기, Q2는 새 영역. 기본 분리 prompt.
2. **Flash 업그레이드 (5/17)** — flash-lite → flash. prompt attention 강화. 토큰 비용 3배지만 11명 규모에선 무시 가능.
3. **prompt 압축 (5/17)** — 5000자 → 1700자. 후반부 attention 살림. 학술 디테일은 docs/research/로 분리.
4. **카테고리 분리 명시 (5/17)** — 영역 tag(일·관계·취미)에서 QueSCo 6 유형(경험탐색·과거성공·본질추출·패턴연결·시점이동·미래가능성)로 기준 변경.
5. **Q2 큐레이션 풀 (5/18)** — prompt 강제론 한계. 구조적 해결: 사용자가 직접 작성한 **139문 풀**에서 Q2를 선택. recency cooldown N=60(약 2달 간격)로 영속 dedup. `qa_pair.question_id` 컬럼 추가(migration 009).

결과: Q1·Q2가 명확히 다른 결로 나옴. 사용자 반응 "질문의 질 확실히 좋아짐".
